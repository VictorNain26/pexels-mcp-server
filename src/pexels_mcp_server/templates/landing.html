Pexels MCP server PUBLIC

Free Pexels photo and video search exposed as a Model Context Protocol server. Nine read-only tools, OAuth 2.1, bring-your-own Pexels API key.

Connect from claude.ai web

  1. Settings → Connectors → Add custom connector.
  2. URL: loading...
  3. Click Connect — the OAuth handshake completes automatically (browser flashes briefly, no input).
  4. Open Advanced settings on the connector and add this custom header: 
    X-Pexels-Api-Key: <your own free Pexels API key>
    Get one at pexels.com/api.
  5. Activate the connector in a conversation, try: « Find me three landscape photos of Paris on Pexels ».

Your Pexels quota stays yours

Your X-Pexels-Api-Key header is forwarded to api.pexels.com on every call and never stored on the server. Every request you make counts against your Pexels quota (200/hour on the free tier), not anyone else's.

Tools exposed

  • Photos: pexels_search_photos, pexels_curated_photos, pexels_get_photo
  • Videos: pexels_search_videos, pexels_popular_videos, pexels_get_video
  • Collections: pexels_list_featured_collections, pexels_get_my_collections, pexels_get_collection_media

Other MCP clients

Same URL works with Claude Desktop (Settings → Connectors → Add), Claude Code (claude mcp add with --transport http), and the official MCP Inspector. OAuth discovery is automatic via RFC 9728 protected-resource metadata.
